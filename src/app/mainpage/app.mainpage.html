How it works?

The Alpha Color Converter is an experimental RGB to RGBA color converter tool to make transparent layering on the web applications. With this tool, you can use transparent fore colors instead of non-alpha fore colors.

The calculation works based on these inputs and table below: RGB1 is fore color, RGB2 is back color and RGB3 is result color.

r3 = r2 + (r1 - r2) * a1
g3 = g2 + (g1 - g2) * a1
b3 = b2 + (b1 - b2) * a1

If your color sets are able to present with alpha value, there can be a large number of RGBA colors (infinite, in floating-point space) that could satisfy the requirements. In this step, the algorithm first checks that the alpha-convertibility of RGB1 and RGB2 color sets. And then It calculates all possible color sets with alpha values and returns the highest possible threshold value in this result list.

Additional to that, there is another calculation for grayscale color. If you directly set two grayscale colors for the fore and background, the algorithm also does the grayscale calculation. If there is a possible solution, you will be informed about this solution as well with an additional section which popups below.

The algorithm works experimental and may not prove the absolute results directly. The result can be the closest value based on your inputs as well. Cause of that, I'm not responsible for any problem which may occur on your project due to this toolchain.
